      causal_engine: CausalEngineImpl,

      belief_engine: BeliefEngine.Interface,
      records: ReadonlyMap<string, AuraRecord>

    **Seed selection algorithm (selectSeeds function):**
    1. Get causal engine state via causalEngine.stats()
    2. Iterate causal state patterns
    3. For each pattern, check 6 gates:
       a. **Strength gate:** pattern.state === "Stable" OR (pattern.state === "Candidate" && pattern.causal_strength >= 0.65)
       b. **Support gate:** pattern.support_count >= 2 OR (evidence_mode === ExplicitTrusted && pattern.explicit_support_count >= 1)
       c. **Evidence gate:** meetsSupportGate && meetsRepeatedEvidenceGate (call functions from CausalEngine)
       d. **Counterevidence gate:** pattern.counterevidence_count <= pattern.support_count / 2 (Rust ratio check)
       e. **Counterfactual gate:** meetsCounterfactualGate from CausalEngine (reuse/inline same logic)
       f. **Belief gate:** beliefEngine.belief_for_record() for each cause-side record → at least one Resolved/Singleton belief exists
    4. Collect passing patterns as seeds

    **Seed structure:**
    ```typescript
    type PolicySeed = {
      pattern: CausalPattern
      cause_belief_id: string | null
      effect_belief_id: string | null
      cause_record_ids: string[]
      effect_record_ids: string[]
    }
    ```

    **aggregateBeliefConfidence function:**
    `(pattern: CausalPattern, records: ReadonlyMap<string, AuraRecord>, beliefEngine: BeliefEngine.Interface) => Effect<number>`
    - For each cause_record_id, look up its belief
    - Average the belief confidences
    - If no beliefs → return 0.5 (default)

    5. Check content against POSITIVE_KEYWORDS → +1 per match
    6. Sum across all effect records

    `(effectRecordIds: string[], records: ReadonlyMap<string, AuraRecord>) => 'Positive' | 'Negative' | 'Neutral'`
