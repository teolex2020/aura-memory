    CRITICAL — ALL formula constants and weights MUST be extracted directly from `@../src/causal.rs` score_pattern. The values below are GUIDES; verify each against Rust source. Do not use placeholder values like "constant ceiling" — every constant must come from Rust.

    **scorePattern function:**
    `(edges: CausalEdge[], records: ReadonlyMap<string, AuraRecord>, allRecords: ReadonlyMap<string, AuraRecord>) => PatternScores`

    Where PatternScores:
    - transition_lift: number (0-1)
    - temporal_consistency: number (0-1)
    - outcome_stability: number (0-1)
    - support_score: number (0-1)
    - confidence_score: number (0-1)
    - causal_strength: number (0-1)
    - significance: number (0-1) — overall significance score

    **transition_lift calculation (verify formula against Rust):**
    - P_effect_given_cause = support_count / cause_records_count
    - P_effect = effect_records_count / total_records_count
    - Lift = (P_effect_given_cause / P_effect)
    - Normalized: min(Lift, X) / X — verify X from Rust (guide says 5.0)
    - If P_effect == 0 → return 0

    **temporal_consistency calculation:**
    - Count edges where gap_seconds > 0
    - total = edges.length
    - consistency = positive_gap_edges / Math.max(1, total)
    - If total == 0 → return 0

    **outcome_stability calculation:**
    - Collect gap_seconds for all temporal edges (gap_seconds > 0)
    - If gaps.length == 0 → return 0
    - mean = avg(gaps)
    - variance = sum((gap - mean)^2) / (gaps.length - 1)  // sample variance
    - stability = 1.0 - (variance / mean)  // Rust formula
    - Clamp to [0, 1] via Math.max(0, Math.min(1, stability))

    **support_score calculation (NO AMBIGUITY — extract from Rust):**
    - ln(1 + support_count) / ln(1 + max_support)  // normalized
    - max_support = maximum support_count across ALL patterns in this cycle (NOT a constant ceiling)
    - If max_support == 0 → return 0
    - Verify ln base from Rust (natural log ln vs log10)

    **confidence_score calculation:**
    - Average confidence of records involved in the causal edges
    - For each edge: avg(confidenceOf(cause_record), confidenceOf(effect_record))
    - confidence_score = avg of all edge confidences

    **causal_strength calculation (weights from Rust — grep for exact values):**
    - causal_strength = w1 * transition_lift + w2 * temporal_consistency + w3 * outcome_stability + w4 * support_score
    - Guide says w1=0.30, w2=0.25, w3=0.25, w4=0.20 — VERIFY each against Rust source
    - If Rust has additional components (significance, confidence_score) in the strength formula, include them
    - Clamp to [0, 1]

    **integration into discover:**
    - After aggregateToPatterns, call scorePattern for each pattern with its edges
    - Populate CausalPattern fields: transition_lift, temporal_consistency, outcome_stability, causal_strength
