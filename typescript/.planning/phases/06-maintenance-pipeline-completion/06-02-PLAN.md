  <behavior>
    The CausalEngine discovers causal patterns from concept engine state.

    **CausalEngine.discover** takes `(concept_state: ConceptEngineState, records: ReadonlyMap<string, AuraRecord>, sdr_lookup: SdrLookup)` and returns `Effect.Effect<CausalReport, never, EpistemicTrace>`.

    **Algorithm (SIMPLE IMPLEMENTATION):**
    1. Get Clock.nowSeconds for timestamps
    2. Get serviceOption(EpistemicTrace) for trace events
    3. Emit trace event "causal.discover.start" with records.size
    4. Build a reverse index: for each record_id, collect all concept_ids that contain it
    5. For each record_id with 2+ concepts (the concepts co-occur on that record), create or score a pair
    6. For each concept pair (A, B), compute support = number of shared records, confidence = support / total records in A
    7. If support > 0, create a CausalPattern with: antecedent_concept_ids=[A], consequent_concept_ids=[B], support, confidence (min(1.0, support / total_records_in_A)), lift (confidence / (B_records / total_records)), state=Candidate or Stable based on confidence > 0.7
    8. Update self.state.patterns — merge new patterns with existing ones
       - Use deterministicId (xxhash-based) pattern for pattern IDs, prefix "cp-"
       - If a pattern already exists, update its confidence/lift but preserve its ID
    9. Build CausalReport from pattern counts
    10. Emit trace event "causal.discover.end" with report fields
    11. Return report

    **CausalEngine.invalidate_pattern(id):**
    - Find pattern by id in self.state.patterns
    - Mark its state as CausalState.Invalidated
    - Return Effect.void

    **CausalEngine.retract_pattern(id):**
    - Delete pattern by id from self.state.patterns
    - Return Effect.void

    **CausalEngine.stats():**
    - Return Effect.succeed(self.state)

    **Test cases:**

    *Test 1: discover with 1 concept returns patterns_found=0*
    Input: ConceptEngineState with 1 concept, 2 records
    Expected: CausalReport with patterns_found=0, patterns_active=0

    *Test 2: discover with 2 concepts sharing records creates patterns*
    Input: ConceptEngineState with 2 concepts, 1 shared record_id between them
    Expected: CausalReport with patterns_found >= 1, patterns_active >= 1

    *Test 3: discover is deterministic across replay*
    Input: Same ConceptEngineState twice
    Expected: Same CausalReport metrics (patterns_found, patterns_active, avg_confidence, avg_lift)

    *Test 4: invalidate_pattern marks pattern as Invalidated*

    function deterministicId(prefix: string, key: string): string {
      // Simple hex hash for deterministic IDs
      let hash = 0
      for (let i = 0; i < key.length; i++) {
        hash = ((hash << 5) - hash + key.charCodeAt(i)) | 0
      }
      return `${prefix}-${Math.abs(hash).toString(16).padStart(8, "0")}`
    }
