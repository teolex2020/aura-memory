  <name>PolicyEngine — discover and retract_hint</name>
  <files>packages/policy/src/PolicyEngine.ts, packages/policy/src/PolicyEngine.test.ts</files>

  <behavior>
    The PolicyEngine extracts policy hints from causal engine state.

    **PolicyEngine.discover** takes `(causal_state: CausalEngineState, records: ReadonlyMap<string, AuraRecord>)` and returns `Effect.Effect<PolicyReport, never, EpistemicTrace>`.

    **Algorithm (SIMPLE IMPLEMENTATION):**
    1. Get Clock.nowSeconds for timestamps
    2. Get serviceOption(EpistemicTrace) for trace events
    3. Emit trace event "policy.discover.start" with info about patterns available
    4. Iterate over patterns in causal_state.patterns
    5. For Stable patterns, create a PolicyHint:
       - id: deterministic hash of pattern.id, prefix "ph-"
       - pattern_id: pattern.id
       - condition: "pattern:${pattern.id}"
       - action: "boost:consequent"
       - priority: Math.round(pattern.confidence * 10)
       - confidence: pattern.confidence
       - state: PolicyState.Stable if pattern.confidence > 0.7 else PolicyState.Candidate
       - last_updated: current timestamp
    6. For patterns with Invalidated state, create Suppressed hints
    7. Merge with existing self.state.hints — preserve existing hint IDs when found
    8. Build PolicyReport:
       - hints_found: total hints created
       - hints_active: hints with Stable state
       - hints_suppressed: hints with Suppressed state
       - avg_confidence: average of hint confidences (0 if none)
    9. Emit trace event "policy.discover.end" with report fields
    10. Return report

    **PolicyEngine.retract_hint(id):**
    - Delete hint by id from self.state.hints
    - Return Effect.void

    **PolicyEngine.stats():**
    - Return Effect.succeed(self.state)

    **Test cases:**

    *Test 1: discover with empty causal state returns hints_found=0*
    Input: CausalEngineState with no patterns
    Expected: PolicyReport with hints_found=0, hints_active=0

    *Test 2: discover with stable patterns creates hints*
    Input: CausalEngineState with 2 Stable patterns at confidence 0.8
    Expected: PolicyReport with hints_found=2, hints_active=2

    *Test 3: discover is deterministic across replay*
    Input: Same CausalEngineState twice
    Expected: Same PolicyReport metrics (hints_found, hints_active, avg_confidence)

    *Test 4: retract_hint removes hint from state*
    Input: Engine state with a hint, call retract_hint(hint_id)
    Expected: stats().hints[hint_id] is undefined

    *Test 5: discover emits trace events*
    Input: Provide EpistemicTrace service
    Expected: trace.event called with "policy.discover.start" and "policy.discover.end"
  </behavior>
