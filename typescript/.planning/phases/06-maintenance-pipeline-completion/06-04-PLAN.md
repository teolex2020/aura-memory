    BoundedReranker.rerank(scored, query) re-scores top-K results.

    **Algorithm (SIMPLE IMPLEMENTATION):**
    The reranker receives a scored list and a query string. It:
    1. Takes the top K results from the input (configurable depth, default 20)
    2. Re-scores each item by combining the original score with an activation boost:
       - boost = min(1.0, activation_count / 10) * 0.1
       - new_score = original_score * (1 + boost)
    3. Returns the re-sorted list preserving the total count

    **Context Requirements:** The reranker needs access to record metadata (activation_count). Since the service is optional and receives only scored tuples `[score, recordId]`, the SIMPLE implementation has no access to records. Instead:
    - Accept the scored list and query (per contract)
    - For SIMPLE: apply a modest score dampening for low-ranked items (inverse position bonus) - this simulates a reranking signal without requiring record access
    - Return re-sorted RecallScored

    **FULL IMPLEMENTATION would:** Use RecallView or another service to fetch record metadata per ID and re-score based on activation_count, confidence, and query similarity.

    Expected: [] as RecallScored

    *Test 3: reranking re-orders by combined score*

    Export `BoundedRerankerImpl` class implementing the rerank method from `@aura/contract`'s `BoundedReranker` Tag shape:
    ```typescript
    import { Effect, Layer, Option } from "effect"
    import { BoundedReranker, type RecallScored } from "@aura/contract"

    const DEFAULT_RERANK_DEPTH = 20
    const INVERSE_POSITION_BOOST = 0.05

    export class BoundedRerankerImpl {
      constructor(private readonly depth: number = DEFAULT_RERANK_DEPTH) {}

      rerank(scored: RecallScored, _query: string): Effect.Effect<RecallScored, never> {
        // SIMPLE IMPLEMENTATION: Apply inverse-position boost to re-score
        // Lower-ranked items get a small boost to simulate diversity-aware reranking
        if (scored.length <= 1) return Effect.succeed(scored)

        const depth = this.depth
        const result: [number, string][] = []
        for (let i = 0; i < scored.length; i++) {
          const [score, id] = scored[i]!
          let newScore = score
          if (i < depth) {
            // Boost lower positions to simulate diversity-aware reranking
            const positionBoost = 1 + INVERSE_POSITION_BOOST * (1 - i / depth)
            newScore = score * positionBoost
          }
          result.push([newScore, id])
        }
        result.sort((a, b) => b[0] - a[0])
        return Effect.succeed(result)
      }
    }

    export const BoundedRerankerLive = Layer.succeed(BoundedReranker, new BoundedRerankerImpl())
    ```

    Export `RecallFinalizerImpl` class:
    ```typescript
    import { Effect, Layer, Console } from "effect"
    import { RecallFinalizer, type RecallScored } from "@aura/contract"

    export class RecallFinalizerImpl {
      private activationCounts = new Map<string, number>()
      private sessions = new Set<string>()

      finalize(scored: RecallScored, sessionId?: string): Effect.Effect<void> {
        for (const [, recordId] of scored) {
          this.activationCounts.set(recordId, (this.activationCounts.get(recordId) ?? 0) + 1)
        }
        if (sessionId) {
          this.sessions.add(sessionId)
        }
        return Effect.log(JSON.stringify({
          type: "recall.finalize",
          records: scored.length,
          sessionId: sessionId ?? null,
          totalActivations: this.activationCounts.size
        }))
      }

      getActivationCount(recordId: string): number {
        return this.activationCounts.get(recordId) ?? 0
      }

      getSessionCount(): number {
        return this.sessions.size
      }
    }

    export const RecallFinalizerLive = Layer.succeed(RecallFinalizer, new RecallFinalizerImpl())
    ```
