    `(records: ReadonlyMap<string, AuraRecord>) => { edges: CausalEdge[], stats: EdgeStats }`

    Edge extraction phases (matching Rust causal.rs:609-730):

    Phase A — Explicit edges (caused_by_id):
    - Iterate records.values()
    - For each rec, if rec.caused_by_id exists AND is in records:
      - If cause and effect are in same namespace (rec.caused_by_id's namespace === rec.namespace):
        - Create CausalEdge: { cause_record_id: caused_by_id, effect_record_id: rec.id, namespace, edge_kind: "explicit", gap_seconds: 0, created_at: rec.created_at }

    Phase B — Explicit edges (connection_type causal):
    - For each rec, iterate rec.connection_types entries
    - If connection_type === "causal" and the connected record exists:
      - Direction determined by created_at (older → newer)
      - Create edge with edge_kind: "explicit_causal"

    Phase C — Temporal edges:
    - Group records by namespace
    - For each namespace, sort records by created_at ASC
    - For each cause_record (i), iterate subsequent records (j > i):
      - gap = records[j].created_at - records[i].created_at
      - If gap > 7 * 86400: break (inner loop — window exceeded)
      - If gap <= 0: continue (same timestamp or ordered wrong — check Rust behavior for ==0 gaps)
      - Create CausalEdge with edge_kind: "temporal", gap_seconds: gap
      - Apply NearbySuccessors budget: increment successor counter per cause; if >= 16, skip remainder

    **aggregateToPatterns function:**
    `(edges: CausalEdge[], records: ReadonlyMap<string, AuraRecord>, beliefEngine: BeliefEngine.Interface) => Effect<Record<string, CausalPattern>, never, EpistemicTrace>`

    Algorithm (matching Rust causal.rs:767-954):
    1. Build reverse mapping: record_id → belief_id via beliefEngine (confirmed contract access path)
       - Only consider beliefs with state Resolved or Singleton (not Unresolved/Empty)
    2. Group edges by cause_belief_id → effect_belief_id pair:
       - For each edge, find cause_record's belief_id and effect_record's belief_id
       - If either belief_id is null → skip edge (no belief mapping)
       - Key = `${cause_belief_id}:${effect_belief_id}`
    3. For each group:
       - If cause_belief_id === effect_belief_id → skip (self-loop filter)
       - Compute support_count = number of edges in group
       - Compute explicit_support_count = edges where edge_kind is "explicit" or "explicit_causal"
       - Compute temporal_consistency = edges with gap_seconds > 0 / total edges in group
       - Compute transition_lift preliminary value
       - Compute causal_strength preliminary value
       - Compute pattern key = `${namespace}:${cause_belief_key}:${effect_belief_key}:${edge_hash}`
       - Generate deterministic pattern ID from pattern key using xxhash
    4. Return map of pattern_id → CausalPattern
