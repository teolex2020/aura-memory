## 06.2-05: Complex Aggregate Methods + Surface Delegation + Aura.runMaintenance

### What was built

Implemented all remaining stubbed methods on `EpistemicRuntimeImpl`:

**Complex Aggregate Methods (4):**
- `getBeliefInstabilitySummary()` — full state counting, volatility bands (low/medium/high), averages, high-volatility and low-stability counts
- `getContradictionClusters(records, namespace?, limit?)` — Union-Find graph via BFS connected components; belief→record mapping from hypotheses; namespace filtering; shared tag computation; sorted by totalConflictMass desc; limit capped at 100
- `getPolicyLifecycleSummary(actionLimit?, domainLimit?)` — aggregation by actionKind (PolicyActionSummary) and namespace×domain (PolicyDomainSummary); advisoryPressure = active/max(1,total); limits clamped to [1,50]
- `getPolicyPressureReport(namespace?, limit?)` — namespace×domain grouping; advisoryPressure, strongest hint tracking; sorted desc; limit capped at 100

**Surface Delegation Methods (5):**
- `getSurfacedConcepts(limit?)` — delegates to `surfaceConcepts()` from @aura/concept
- `getSurfacedConceptsForNamespace(namespace, limit?)` — delegates to `surfaceConceptsFiltered()` from @aura/concept
- `getSurfacedConceptsForRecord(recordId, limit?)` — inline filter by record_id in concept's record_ids; default limit 3
- `getSurfacedPolicyHints(limit?)` — builds adapter from contract PolicyEngine → surface PolicyEngine; delegates to @aura/policy
- `getSurfacedPolicyHintsForNamespace(namespace, limit?)` — same adapter pattern with namespace filter

**Aura.ts:**
- Added `runMaintenance()` stub using `Effect.die(new UnimplementedError(...))`

**Dependencies:**
- Added `@aura/concept: workspace:*` and `@aura/policy: workspace:*` to @aura/epistemic-runtime

### Implementation notes

- Volatility bands: low < 0.20, medium 0.20–0.50 (inclusive), high > 0.50
- Contradiction clusters: O(n²) belief pair comparison for edge detection; BFS for connected components; namespace derived from records associated with belief hypotheses
- Policy surface adapter: converts contract PolicyHint (mixed snake_case/camelCase) to surface module PolicyHint; eligible hints pass through surface module's filtering pipeline
- Helper functions: `computeSharedTags`, `toSurfacePolicyHint`, `toContractSurfacedPolicyHint`
